Requerimientos Funcionales


- Requerimiento #1

        * Visualización del ganado registrados

        Descripción:
            Se debe permitr por medio del sistema de información visualizar los datos del ganado registrado.

        Historia de Usuario:

        Como funcionario publico, quiero visualizar la información del ganado registrado, 
        para informarme acerca de su porpietario y ubicación.
        Criterio de aceptación:
		El sistema de información mostrara una tabla de todo el ganado registradoa con su respectiva información.



- Requerimiento #2

        * Visualización de las zonas afectadas por la deforestación

        Descripción:
            Se debe permitr por medio del sistema de información visualizar las zonas afectadas por la deforestación

        Historia de Usuario:

        Como funcionario publico, quiero visualizar la información de las zonas deforestadas que se han registrado, 
        para informarme acerca de las zonas donde se debe realizar las campañas de pedagogicas.
        Criterio de aceptación:
		El sistema de información mostrara un mapa de las zonas afectadas.

- Requerimiento #3

        * Visualización del ganado que se encuentra dentro de las zonas de deforestación

        Descripción:
            Se debe permitr por medio del sistema de información visualizar el ganado afectado por las zonas de deforestación

        Historia de Usuario:

        Como funcionario publico, quiero visualizar la información el ganado afectado por las zonas de deforestación, 
        para informarme conocer el ganado afectado.
        Criterio de aceptación:
		El sistema de información mostrara un mapa de las zonas afectadas con una marca de cada animal con su propietario.


Requerimientos no Funcionales

    - Facilidad de despliegue.
        Se debe guardar una imagen de docker para facilitar el despliegue.
    - Requisitos de software
        El front-end se desarrollara en React JS en Vite
        El back-end se desarrollara en Node JS con Express
        Acceso a un navegador web (Chrome, Firefox, Edge, etc)
    - Requisitos de hardware
        Se requiere acceso a internet.
        Se requiere un dispositivo con acceso a un navegador.



Alcance del aplicativo:


    Objetivos del proyecto: 
        
        Con el proyecto se le dara la posibilidad a las entidades gubernamentales de realizar la revisión de la información
        de el ganado y zonas afectadas por la deforestación, asi mismo se podra visualizar un mapa donde 
        encontrara el ganado dentro de estas zonas.


    Entregables:

        Visualización de una tabla con información del ganado y las zonas.
        Visualización de un mapa con las marcas o zonas del ganado o zona seleccionada.
        Visualización de un mapa con las zonas afectadas por la deforestación y los puntos donde se encuentran los animales.
        Login de acceso al aplicativo.
    

    Fuera del alcance:

        Posiblidad de adicionar información de nuevo ganado o zonas de deforestación.
        Capa de seguridad de los servicios.
        Filtros para la información.
